sip_settings.autodomain = Auto-Domäne
sip_settings.autodomain.tooltip =
	Dies ist eine wichtige Einstellung in Bezug auf SIP-Domänen. Wenn es auf "nein" gesetzt ist, erkennt Asterisk nur
	Domänen, die explizit definiert wurden oder einfach keine SIP-Domänen unterstützen (wenn es keine explizit definierten Domänen gab).
	Wenn Sie es auf "ja" setzen, beachten Sie bitte, dass Asterisk eine Domain basierend auf der externen IP-Adresse Ihres
	Firewall wie im Parameter "externip" angegeben. Dies kann eine Gefährdung Ihrer SIP-Sicherheit darstellen.
	Wenn Sie nicht möchten, dass eine Domain basierend auf "externip" erstellt wird, dann setzen Sie auf "no" und fügen Sie explizit Domains hinzu
	für Ihre lokale (interne) IP-Adresse und für alle anderen von Ihnen benötigten Domains.

sip_settings.sip_domains = SIP-Domänen
sip_settings.sip_domains.info =
	Wenn sie verwendet werden, bieten sie erhöhte Sicherheit, da Registrierungen nur akzeptiert werden, wenn sie von einem IP-Telefon kommen
	(oder ein anderer SIP-Client), der eine der erkannten Domänen verwendet. Wenn Asterisk die Identität aller seiner lokalen
	SIP-Domains ermöglicht dies auch eine höhere Sicherheit beim Routing von SIP-to-SIP-Anrufen
