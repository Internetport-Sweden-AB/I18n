dpma_general_settings.auth_option = Options d'authentification
dpma_general_settings.config_auth = Authentification de la configuration
dpma_general_settings.config_auth.disabled = Désactivé
dpma_general_settings.config_auth.globalpin = PIN global
dpma_general_settings.config_auth.mac = Mac
dpma_general_settings.config_auth.pin = Broche
dpma_general_settings.config_auth.tooltip = Spécifie quel type d'authentification est requis pour qu'un téléphone récupère une configuration valide du serveur pour un utilisateur. Vous trouverez ci-dessous les valeurs valides pour cette option.
dpma_general_settings.config_auth.tooltip.disabled = Les configurations sont servies sans authentification. N'importe quel téléphone peut extraire n'importe quelle configuration de téléphone définie dans ce fichier sans aucun défi d'authentification.
dpma_general_settings.config_auth.tooltip.globalpin = Seul le mot de passe global (ou group_pin, s'il est défini sur un téléphone) doit être valide pour récupérer la configuration.
dpma_general_settings.config_auth.tooltip.mac = L'adresse MAC seule doit correspondre à la configuration du téléphone demandée.
dpma_general_settings.config_auth.tooltip.pin = Seul le mot de passe spécifique du téléphone doit être valide pour récupérer la configuration demandée.
dpma_general_settings.file_directory = Répertoire de fichiers
dpma_general_settings.file_directory.tooltip = Le répertoire de fichiers pointe vers le dossier que les téléphones sont autorisés à consulter pour récupérer les fichiers txt et binaires. Par défaut, ce répertoire se trouve dans /var/lib/asterisk/digium_phones.
dpma_general_settings.general = Général
dpma_general_settings.globalpin = Broche globale
dpma_general_settings.globalpin.tooltip = Si userlist_auth est défini sur globalpin, définit le code PIN qui doit être saisi sur un téléphone, au démarrage, afin de récupérer la liste des configurations de téléphone disponibles. Si config_auth est défini sur globalpin, définit le code PIN qui doit être entré sur un téléphone, à l'écran de liste d'utilisateurs, afin de demander une configuration de téléphone particulière - notez que si le globalpin a déjà été entré pour s'authentifier pour récupérer la liste des téléphones disponibles configurations, qu'il n'aura pas besoin d'une seconde fois pour demander une configuration téléphonique particulière. Si group_pin est utilisé pour les téléphones et que userlist_auth et config_auth sont définis sur globalpin, alors l'entrée du globalpin affichera tous les téléphones pour tous les groupes.
dpma_general_settings.mdns = Options de découverte de service mDNS
dpma_general_settings.mdns_address = Adresse d'enregistrement
dpma_general_settings.mdns_address.tooltip = L'adresse d'enregistrement fournie par le DPMA a Avahi. Il s'agit de l'adresse à laquelle les téléphones découvrant le DPMA à l'aide de mDNS seront contactés.
dpma_general_settings.mdns_port = Port d'enregistrement
dpma_general_settings.mdns_port.tooltip = Le port d'enregistrement fourni par le DPMA à Avahi. Il s'agit du port auquel les téléphones découvrant le DPMA à l'aide de mDNS se connecteront.
dpma_general_settings.other_options = Autres options
dpma_general_settings.pjsip_message_context = Contexte du message PJSIP
dpma_general_settings.pjsip_message_context.tooltip = Le contexte utilisé pour router les messages dans pjsip afin que les conflits trouvés dans pjsip.conf puissent être évités. La valeur par défaut est dpma_pjsip_message_context.
dpma_general_settings.server_uuid = Identificateur de serveur
dpma_general_settings.server_uuid.tooltip = Définit un identifiant unique pour ce serveur. Utilisé, notamment, au sein de l'application de messagerie vocale pour le renvoi vers des contacts associés à ce serveur. Facultatif, pas de valeur par défaut. Si server_uuid est défini dans la section générale, il doit également être défini pour les contacts individuels. S'il n'est défini qu'à un seul emplacement (général ou contact) au lieu des deux, le transfert des messages vocaux aux contacts n'est pas autorisé.
dpma_general_settings.service_discovery_enabled = Découverte de service
dpma_general_settings.service_discovery_enabled.tooltip = Annoncer ou non le serveur de configuration en utilisant Avahi. Par défaut, cette option est définie sur oui. Le service de découverte ne sera activé que lorsque le serveur d'enregistrement et le port sont explicitement configurés.
dpma_general_settings.service_name = Nom du service
dpma_general_settings.service_name.placeholder = Serveur de configuration DPMA
dpma_general_settings.service_name.tooltip = Le nom du service à utiliser dans la découverte de service mDNS. Ce sera par défaut "Digium Phones Config Server" lorsque Service Discovery est défini et que l'adresse et le port d'enregistrement sont présents.
dpma_general_settings.userlist_auth = Authentification de la liste des utilisateurs
dpma_general_settings.userlist_auth.disabled = Désactivé
dpma_general_settings.userlist_auth.globalpin = PIN global
dpma_general_settings.userlist_auth.tooltip = Spécifie quel type d'authentification est requis pour qu'un téléphone extrait la liste des utilisateurs. Si vous définissez ce paramètre sur "Global Pin", l'utilisateur du téléphone doit saisir la valeur globalpin avant d'être autorisé à obtenir la liste des utilisateurs.
